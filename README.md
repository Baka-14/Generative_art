# Generative Art (Processing)

Small collection of generative art experiments built in Processing, inspired by *The Nature of Code*. Each sketch explores a different physics or motion concept like oscillation, attraction, gravity, and rotation.

## Project structure

- `sketches/` – all Processing `.pde` sketches
  - `screen_saver.pde` – oscillation-based screen-saver style animation
  - `Atom_1.pde` – atom-like structure using radial motion
  - `genart4.pde` – bouncing balls with gravity and changing colours (“trippy balls”)
  - `gen_art_one.pde` – particles affected by gravity and wind
  - `gen_art_astrik.pde` – gravitational attraction creating starburst patterns
  - `the_sun.pde` – rotating radial orbits
  - `Flash.pde` – simple rotational motion study

## How to run

- **1. Install Processing**: Download Processing from `https://processing.org`.
- **2. Open a sketch**: In Processing, open any `.pde` file from the `sketches/` folder.
- **3. Run**: Press the **Run** button ▶ to see the animation.
